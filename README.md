# 🧠 Task Management System - Backend API

This is the backend REST API for the Task Management System built using Node.js and Express.js.  
It handles authentication, user management, and task CRUD operations.

---

## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/Hansana-Sandamini/Task-Management-System-Backend
```

### 2. Install dependencies
```bash
npm install
```

### 3. Create `.env` file
```bash
PORT=5000
MONGO_URI=your_database_url
JWT_SECRET=your_jwt_secret_key
JWT_REFRESH_SECRET=your_jwt_refresh_secret_key
NODE_ENV=development
FRONTEND_URL=deployed_frontend_url
```

### 4. Run the server
```bash
npm run dev
```
